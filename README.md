## Instalación

Necesitas tener instalado Python 3.8 en tu sistema, usando pyenv o cualquier otro método. Tras activarlo con `pyenv global 3.8.2`, por ejemplo, escribe

    pipenv install
	
Y a continuación

    pipenv shell
	
Para activarlo.
